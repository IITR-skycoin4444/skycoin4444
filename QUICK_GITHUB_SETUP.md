# Quick GitHub Enterprise + GitHub Pages Setup

## 🚀 3-Minute Setup

### Step 1: Connect GitHub Enterprise (1 minute)

**In Manus Management UI:**
1. Click **More** (⋯) → **GitHub**
2. Select **GitHub Enterprise**
3. Enter your enterprise domain
4. Click **Authorize** and sign in
5. Grant permissions

### Step 2: Export Project (1 minute)

**In Manus Management UI:**
1. Click **More** (⋯) → **GitHub**
2. Select your organization
3. Enter repo name: `skycoin4444`
4. Click **Export**

### Step 3: Enable GitHub Pages (1 minute)

**In GitHub Enterprise:**
1. Go to your repository
2. Click **Settings** → **Pages**
3. Source: **Deploy from a branch**
4. Branch: **gh-pages**
5. Click **Save**

## 🆘 Troubleshooting

**Site shows 404?**
- Check GitHub Pages is enabled
- Verify branch is `gh-pages`
- Wait 1-2 minutes for deployment

**Build failed?**
- Check Actions tab for error logs
- Run `pnpm build` locally to test

**Need custom domain?**
- Add domain in GitHub Pages settings
- Configure DNS CNAME record
- Wait 24-48 hours for DNS propagation
